GitHub上邀请Collaborators

1.选择repository
2.选择repository的setting里面的collaborators，搜索你要添加的collaborator，点击Add collaborator添加

3.完成后，会有一封邮件发送到对方注册GitHub账号时的时候使用的邮箱里
4.对方通过邮箱确认，接受邀请（邮件有可能被拦截到垃圾邮箱）
5.邀请成功后，被邀请者就可以获得push权限了
